# TFE – Multimodal Physiological Modeling of The Trombonist's Musical Comfort

## Auteur
Denis Albrecq

Date de mise à jour : 15 février 2026

## Objectif du projet
Ce projet vise à mesurer et modéliser le confort de jeu des trombonistes en combinant :
- Signaux physiologiques : HRV, GSR, respiration
- Signaux non-biologiques : audio, gestuelle
- Validation subjective : questionnaire K-MPAI, STAI-S, NASA-TLX
